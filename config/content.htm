2 先登入Github到老師網頁的About點選網址進入後按Use this template，在Repository name輸入自己需要的倉儲名稱後按Create repository from template

3 進自己的倉儲後，按上方的Settings->Pages->None改main->Save->生 成網頁網址

4 開下載好的ipv6，將 ipv4拉進白窗，將 Python38都改成 Python395、 portablegit 改成 portablegit2.31.1，最下面將改成這樣

  git clone --recurse-submodules https://...新建的倉儲網址.git， 跑完之後會發現data->tmp->多 了倉儲名稱的資料夾

6 小黑窗輸稱 

  cd 倉儲名稱 

  pip install pyopenssl 

  cms=>複製網址進入編輯的地方，預設登入密碼為“admin ”

7 回到倉儲(Github)右上角下拉式選單按 Settings->Developer settings->Personal access tokens->Generate new token-> Note輸入 認得出來的名字-> Expiration可以選No expiration(無期限)->rope一 定要勾-> Generate token->出現一大串英文是新建的token要先複製另 外存好
